# UniProton 项目结构参考

## 主要目录结构

- `UniProton/` - 项目根目录
- `UniProton/kernel/` - 内核源代码
- `UniProton/bsp/` - Board Support Package (板级支持包)
- `UniProton/demos/` - 示例应用程序
- `UniProton/demos/riscv64virt/` - RISC-V 64位虚拟平台示例
- `UniProton/demos/riscv64virt/build/` - RISC-V 64位构建目录
- `UniProton/lib/` - 库文件
- `UniProton/config/` - 配置文件

## 常见示例应用

- `hello_world` - 基础示例
- `task_manage` - 任务管理示例
- `sem_test` - 信号量测试示例
- `mutex_test` - 互斥锁测试示例
- `queue_test` - 队列测试示例
- `timer_test` - 定时器测试示例
