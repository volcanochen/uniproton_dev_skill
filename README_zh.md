# UniProton RISC-V 应用开发技能

## 此技能的功能

该技能帮助用户在 Docker 环境中开发、编译和运行 UniProton RISC-V 应用程序。它提供了完整的工作流程，包括：
- 设置 UniProton 开发环境
- 编译 UniProton 应用程序
- 运行和调试 UniProton 应用
- 解决开发过程中的常见问题

## 如何调用此技能

要使用此技能，只需在您的请求中提及您的 UniProton RISC-V 应用开发需求。例如：

```
我需要帮助开发 UniProton RISC-V 应用程序
我想编译并运行 UniProton 应用
你能帮我设置 UniProton 开发环境吗？
```

该技能将引导您完成整个过程，从环境设置到应用部署。

## 主要功能

- **基于 Docker 的环境**：使用预配置的 Docker 镜像，包含所有必要的工具
- **简化的工作流程**：通过清晰的步骤简化开发过程
- **QEMU 集成**：提供内置支持，用于运行和测试应用程序
- **全面的故障排除**：帮助解决常见的开发问题
- **结构化操作日志**：维护所有操作的详细记录

## 何时使用此技能

当您需要以下操作时使用此技能：
- 开发新的 UniProton RISC-V 应用程序
- 编译和测试现有的 UniProton 应用程序
- 配置 UniProton 开发环境
- 排查 UniProton 应用程序问题

## 技能信息

- **技能名称**：uniproton-app-dev
- **描述**：帮助用户在 Docker 环境中开发、编译和运行 UniProton RISC-V 应用程序
- **主要用例**：应用开发、环境配置、编译和测试
